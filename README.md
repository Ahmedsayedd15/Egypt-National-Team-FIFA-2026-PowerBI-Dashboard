# 🇪🇬 Egypt National Team FIFA World Cup 2026 - Power BI Dashboard

## Overview
An interactive Power BI dashboard analyzing the performance of the Egypt National Team during the FIFA World Cup 2026.

The project transforms football data into meaningful insights using data modeling, DAX calculations, and interactive visualizations.

## Dashboard Features
- Executive Overview
- Player Performance Analysis
- Team Performance Analysis
- Goalkeeper Statistics
- Goal & Assist Analysis
- Physical Performance Analysis
- Tournament Journey

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Excel

## Dataset
The dataset includes:
- Player statistics
- Match performance
- Goals & assists
- Physical metrics
- Team statistics
